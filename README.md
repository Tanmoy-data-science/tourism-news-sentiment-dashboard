# Tourism News Sentiment Dashboard

This repository contains an R Shiny dashboard for analyzing
media narratives around tourism recovery.

## Features
- Topic modeling visualization
- Sentiment timeline
- State mentions analysis
- Word cloud
- Article browser

## Requirements
R packages needed:

shiny
bslib
tidyverse
readr
plotly
DT
viridis
wordcloud

## Run the app

In R:

shiny::runApp("app.R")
